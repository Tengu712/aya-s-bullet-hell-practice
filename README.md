## Development

オブジェクト指向プログラミングパラダイムに従っています。勉強も兼ねて、過剰に従っています。

ディレクトリを以下の二つに分けています。

- `interfaces`
  - `@/`でアクセス可能
  - インターフェースとユーザ定義型
  - 必ず、`import`はこのディレクトリ内でのみ完結する
- `sources`
  - `~/`でアクセス可能
  - 具象クラス
  - 原則、`import`は`interfaces`内か`sources`内のインターフェースとユーザ定義型のみに限る

規則として、以下を定めます。

- 具象クラスが一つしか想定されないオブジェクトでも必ずインターフェースを介す
- 以下のクラスでは、例外的にインスタンスの生成を許し、生成のために`import`を許す
  - メインクラス
  - ファクトリクラス
  - シーンクラス
